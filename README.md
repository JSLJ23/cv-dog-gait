# Dog Gait Analysis

Dog gait analysis software for video-based pose estimation, fine-tuning, annotated overlays, and limb-angle tracking.

## Quick Start

This project uses `uv` for Python package management.

```bash
uv sync --dev
uv run streamlit run app.py
```

Run tests:

```bash
uv run pytest
```
